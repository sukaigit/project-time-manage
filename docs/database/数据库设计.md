# 数据库设计

## 数据库
```sql
CREATE DATABASE IF NOT EXISTS project_time_manage DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

## 表结构

### 1. 用户表 — tb_user

| 字段 | 类型 | 约束 | 说明 |
|:-----|:-----|:----|:-----|
| id | BIGINT | PK, AUTO_INCREMENT | 主键 |
| username | VARCHAR(50) | UNIQUE, NOT NULL | 用户名 |
| password | VARCHAR(255) | NOT NULL | BCrypt 加密 |
| name | VARCHAR(50) | NOT NULL | 姓名 |
| dept | VARCHAR(100) | DEFAULT '研发与交付中心' | 所属部门 |
| status | TINYINT | DEFAULT 1 | 1=启用, 0=停用 |
| first_login | TINYINT | DEFAULT 1 | 1=首次登录未改密 |
| create_time | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| update_time | DATETIME | ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

### 2. 角色表 — tb_role

| 字段 | 类型 | 约束 | 说明 |
|:-----|:-----|:----|:-----|
| id | BIGINT | PK, AUTO_INCREMENT | 主键 |
| name | VARCHAR(50) | NOT NULL | 角色名称 |
| code | VARCHAR(50) | UNIQUE, NOT NULL | 角色编码（admin/dept_manager/project_manager/employee） |
| status | TINYINT | DEFAULT 1 | 1=启用, 0=停用 |
| note | VARCHAR(255) |  | 备注 |
| create_time | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |

### 3. 用户角色关联表 — tb_user_role

| 字段 | 类型 | 约束 | 说明 |
|:-----|:-----|:----|:-----|
| id | BIGINT | PK, AUTO_INCREMENT | 主键 |
| user_id | BIGINT | FK → tb_user.id, NOT NULL | 用户ID |
| role_id | BIGINT | FK → tb_role.id, NOT NULL | 角色ID |

UNIQUE(user_id, role_id)

### 4. 角色权限表 — tb_role_permission

| 字段 | 类型 | 约束 | 说明 |
|:-----|:-----|:----|:-----|
| id | BIGINT | PK, AUTO_INCREMENT | 主键 |
| role_id | BIGINT | FK → tb_role.id, NOT NULL | 角色ID |
| menu_key | VARCHAR(50) | NOT NULL | 菜单模块标识（dashboard/project/task/...） |
| actions | VARCHAR(500) | NOT NULL | JSON数组，如 ["查看","新建","编辑"] |

UNIQUE(role_id, menu_key)

### 5. 项目表 — tb_project

| 字段 | 类型 | 约束 | 说明 |
|:-----|:-----|:----|:-----|
| id | BIGINT | PK, AUTO_INCREMENT | 主键 |
| name | VARCHAR(100) | NOT NULL | 项目名称 |
| code | VARCHAR(50) | UNIQUE, NOT NULL | 项目编号 |
| description | VARCHAR(500) |  | 项目描述 |
| dept | VARCHAR(100) | DEFAULT '研发与交付中心' | 所属部门 |
| status | TINYINT | DEFAULT 1 | 1=启用, 0=停用 |
| start_date | DATE |  | 开始日期 |
| end_date | DATE |  | 结束日期 |
| create_by | BIGINT | FK → tb_user.id | 创建人 |
| create_time | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |

### 6. 项目成员表 — tb_project_member

| 字段 | 类型 | 约束 | 说明 |
|:-----|:-----|:----|:-----|
| id | BIGINT | PK, AUTO_INCREMENT | 主键 |
| project_id | BIGINT | FK → tb_project.id, NOT NULL | 项目ID |
| user_id | BIGINT | FK → tb_user.id, NOT NULL | 成员ID |

UNIQUE(project_id, user_id)

### 7. 任务表 — tb_task

| 字段 | 类型 | 约束 | 说明 |
|:-----|:-----|:----|:-----|
| id | BIGINT | PK, AUTO_INCREMENT | 主键 |
| name | VARCHAR(100) | NOT NULL | 任务名称 |
| code | VARCHAR(50) | UNIQUE, NOT NULL | 任务编号 |
| project_id | BIGINT | FK → tb_project.id, NOT NULL | 所属项目 |
| status | TINYINT | DEFAULT 1 | 1=启用, 0=停用 |
| create_by | BIGINT | FK → tb_user.id | 创建人 |
| create_time | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |

### 8. 工时记录表 — tb_time_entry

| 字段 | 类型 | 约束 | 说明 |
|:-----|:-----|:----|:-----|
| id | BIGINT | PK, AUTO_INCREMENT | 主键 |
| user_id | BIGINT | FK → tb_user.id, NOT NULL | 填报人 |
| project_id | BIGINT | FK → tb_project.id, NOT NULL | 项目 |
| task_id | BIGINT | FK → tb_task.id | 任务（项目经理可不填） |
| work_date | DATE | NOT NULL | 工作日期 |
| hours | DECIMAL(4,1) | NOT NULL | 时长（0.5~24） |
| content | VARCHAR(500) |  | 工作内容 |
| status | TINYINT | DEFAULT 0 | 0=待审批, 1=已通过, 2=已驳回 |
| reject_reason | VARCHAR(500) |  | 驳回原因 |
| create_time | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| update_time | DATETIME | ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

INDEX(user_id), INDEX(project_id), INDEX(work_date), INDEX(status)

### 9. 审批记录表 — tb_approval_log（可选，备查）

| 字段 | 类型 | 约束 | 说明 |
|:-----|:-----|:----|:-----|
| id | BIGINT | PK, AUTO_INCREMENT | 主键 |
| time_entry_id | BIGINT | FK → tb_time_entry.id, NOT NULL | 工时记录ID |
| approver_id | BIGINT | FK → tb_user.id, NOT NULL | 审批人 |
| action | TINYINT | NOT NULL | 1=通过, 2=驳回 |
| reason | VARCHAR(500) |  | 驳回原因 |
| create_time | DATETIME | DEFAULT CURRENT_TIMESTAMP | 审批时间 |
