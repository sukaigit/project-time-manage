# 项目工时管理系统 — Phase 1 原型调整记录

## 原型阶段做的需求调整（已同步更新 OpenSpec）

### 模块名变更
| 原名称 | 新名称 |
|:------|:-------|
| 工时填报 | 工时管理 |
| 工时统计 | 员工工时统计 + 项目工时统计（拆为两个子菜单） |

### 功能调整
| 模块 | 调整内容 |
|:-----|:---------|
| 项目管理 | 新增「所属部门」字段（默认研发与交付中心） |
| 项目管理 | 新增「项目成员管理」——通过成员管理弹窗勾选/取消项目成员 |
| 任务管理 | 去掉「分配人员」字段，任务仅作为项目下的工作拆分单元 |
| 工时管理 | 选项目后，任务下拉仅显示该项目下的任务（按项目过滤） |
| 工时管理 | 仅已驳回的记录可编辑重提，待审批/已通过不可编辑 |
| 工时审批 | 驳回时需填写驳回原因（textarea 必填输入框） |
| 工时审批 | 按角色过滤：部门经理→审批项目经理，项目经理→审批普通员工 |
| 工时统计 | 拆为两个独立页面：员工工时统计（姓名/角色/本月/累计）、项目工时统计（项目名/负责人/部门/本月/累计） |
| 工时统计 | 均为列表式展示，不做排名，不按角色区分数据 |
| 首页仪表盘 | 新增任务总数卡片、项目/员工TOP5（月度+年度） |
| 首页仪表盘 | 去掉待审批卡片，近期工时动态增加任务列 |
| 角色管理 | 权限树菜单名同步更新（工时填报→工时管理，工时统计拆为两个） |
| 角色管理 | 权限树父子联动：勾选父级自动勾选子级，取消子级父级半选 |
| 用户管理 | 新增重置密码（默认 uu888888）、删除功能 |
| 用户管理 | 角色下拉从角色管理数据动态加载 |
| 任务管理 | 分配人员改为从用户列表勾选（支持模糊搜索） |
| 用户管理 | 查询条件角色下拉从角色管理动态加载 |
